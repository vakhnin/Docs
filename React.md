# [Docs](README.md) / React

Диспетчер версий
```commandline
nvm
```

Установка определенного пакета (axios)
```commandline
npm install axios
```

Установка зависимиостей из package.json
```commandline
npm install
```

Запуск проекта
```commandline
npm start
```
Список зависимостей
```commandline
npm ls
```
